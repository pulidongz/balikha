# Design Context

`PRODUCT.md` at the project root holds the strategic design context: target
users, product purpose, brand personality, anti-references, and design
principles. `DESIGN.md` holds the visual system (the "Sampaguita & Sea" tokens,
typography, and components).

The default register is **brand** (the editorial storefront). Internal app
surfaces (`app/(dashboard)`, `app/(account)`, `app/(admin)`) are **product**
register. Read PRODUCT.md and DESIGN.md before any design work.
